# Sherpa-ONNX Skill Creation Plan

## Mục tiêu
Tạo Kiro skill hướng dẫn làm việc với sherpa-onnx (k2-fsa/sherpa-onnx) — thư viện offline speech processing dùng ONNX Runtime.

## Scope
- Speech-to-text (streaming + non-streaming)
- Text-to-speech (Piper, Kokoro, Matcha, VITS, KittenTTS)
- Speaker diarization, identification, verification
- VAD (silero-vad)
- Speech enhancement, source separation
- Audio tagging, keyword spotting
- Python API focus (primary), CLI secondary
